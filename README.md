## ‚ú® Features

- **Symmetric Bento Layout**: A modern, clean dashboard design optimized for clarity.
- **Real-Time Search**: Instant city search with arrow-key navigation and "Enter" to select.
- **Hourly Timeline**: Visual 24-hour forecast with dynamic weather icons.
- **Detailed Metrics**: Humidity, wind speed, feels-like temperature, and comprehensive Air Quality Index (AQI).
- **AI Summary**: A personalized, human-readable weather overview.
